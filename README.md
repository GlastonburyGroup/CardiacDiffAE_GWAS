# Hundreds of cardiac MRI traits derived using 3D diffusion autoencoders share a common genetic architecture

[![DOI](https://img.shields.io/badge/DOI-10.1038%2Fs41467--026--74575--y-blue)](https://doi.org/10.1038/s41467-026-74575-y)

The official code for the paper, "Hundreds of cardiac MRI traits derived using 3D diffusion autoencoders share a common genetic architecture", published in [Nature Communications](https://www.nature.com/articles/s41467-026-74575-y). This repository contains all scripts used in this project, except for the deep learning pipeline (DL-pipeline). It includes scripts for pre-processing raw datasets from the UK Biobank, post-processing and analysing the latent embeddings, as well as conducting downstream analyses. The deep learning pipeline for unsupervised latent phenotyping using the 3D diffusion autoencoder can be found here: https://github.com/GlastonburyGroup/ImLatent.
