# UK Biobank Field 20208: Cardiac CINE Long Axis Dataset Creation Guide

## Overview

This guide provides step-by-step instructions for creating and processing the UK Biobank Field 20208 dataset (Long axis heart images - DICOM). The dataset consists of cardiac CINE MRI scans in DICOM format, packaged as ZIP files, which need to be converted into HDF5 format for further analysis.

## Dataset Information

- **Field ID**: 20208
- **Field Name**: Long axis heart images - DICOM (F20208_Long_axis_heart_images_DICOM)
- **Data Format**: ZIP files containing DICOM series
- **Output Format**: HDF5 (.h5) files with organised cardiac imaging data
